# Loganayaki

## Sri Loganayaki Clear Filing Solutions
A premium, editorial-grade financial services website built with modern CSS, smooth animations, and a focus on clarity and professional growth.

### Features
- **Modern UI/UX**: Cinematic hero section with fluid typography and mesh gradients.
- **25+ Years of Experience**: Highlighted "About Us" section with corporate heritage.
- **Comprehensive Services**: GST, Tax, Audit, and Payroll solutions.
- **Fully Responsive**: Optimized for mobile, tablet, and desktop.
- **High Performance**: Vanilla HTML/CSS with no heavy dependencies.

### Technologies
- HTML5
- CSS3 (Custom Design System)
- JavaScript (Scroll Reveal & Mobile Interactions)

### Setup
Simply open `index.html` in any modern web browser.
